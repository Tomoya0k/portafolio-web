# Portafolio Web

Base profesional para portafolio personal con React + Vite + TypeScript + TailwindCSS.

## Requisitos

- Node.js 18+

## Instalacion

```bash
npm install
npm run dev
```

## Estructura

```text
src/
  components/
  sections/
  layout/
  assets/
  data/
  hooks/
  types/
  pages/
```

## Scripts

- `npm run dev`: inicia entorno de desarrollo
- `npm run build`: genera build de produccion
- `npm run preview`: previsualiza build local
